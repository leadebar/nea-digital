# Néa Digital

Site Next.js 15 pour Néa Digital : boutique digitale, pages de vente, services, blog SEO et structure prête pour Vercel.

## Lancer le projet

```bash
pnpm install
pnpm dev
```

## Déployer

Le projet est prêt pour Vercel avec App Router, TypeScript, TailwindCSS, Framer Motion, sitemap et robots générés par Next.js.

## Fonts

La direction artistique utilise `Monument Extended`, `Bebas Neue` et `TT Commons Pro` dans la hiérarchie CSS. Pour un rendu final parfaitement fidèle, ajoute les fichiers de police licenciés dans `public/fonts` puis déclare-les dans `app/globals.css`.
